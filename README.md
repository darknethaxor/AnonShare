## Installation : 

Note: You must be clone in home directory
 
* `apt update` 
* `apt upgrade -y` 
* `apt install git -y`
* `apt install wget -y` 
* `cd $HOME` 
* `git clone https://github.com/darknethaxor/AnonShare.git` 
* `cd AnonShare` 
* `chmod +x share.sh` 
* `bash share.sh`
